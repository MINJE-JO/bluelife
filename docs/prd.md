# 연간 프로젝트 플래너 (Project Blueprint)

## 1. 제품 개요

연간/월간 단위로 프로젝트를 시각적으로 관리할 수 있는 웹 애플리케이션입니다. 사용자는 직관적인 드래그 앤 드롭 인터페이스를 통해 프로젝트 일정을 관리하고, 색상 코딩으로 프로젝트를 구분할 수 있습니다.

### 주요 특징
- 연도/월 기반의 타임라인 뷰
- 드래그 앤 드롭으로 쉬운 일정 관리 
- 실시간 데이터 동기화
- 반응형 웹 디자인

## 2. 핵심 기능

### 2.1 메인 테이블 UI

#### 구조
- 12개 행(월) × N개 열(연도)의 그리드 레이아웃
- 각 셀에 프로젝트 정보 표시
- 프로젝트별 고유 색상 구분
- 호버 시 프로젝트 상세 정보 툴팁

#### 상호작용
- 드래그 앤 드롭으로 프로젝트 기간 조정
- 드래그 중인 셀 범위 하이라이트
- 실시간 날짜 업데이트

### 2.2 프로젝트 관리

#### 프로젝트 생성
- 모달 창을 통한 프로젝트 정보 입력
- 필수 정보: 이름, 시작일, 종료일, 색상
- 즉시 테이블에 반영

#### 프로젝트 수정
- 드래그로 기간 수정
- 클릭으로 상세 정보 수정
- 실시간 저장

## 3. 기술 스택

### Frontend
- **Next.js 15**
  - SSR/CSR 하이브리드 렌더링
  - App Router 사용
- **Clerk**
  - 사용자 인증 및 관리
  - 소셜 로그인 (Google, GitHub 등)
  - 보안 및 세션 관리
- **Shadcn UI**
  - 모던한 UI 컴포넌트
  - 테이블, 모달, 컬러피커 등
- **TailwindCSS**
  - 반응형 디자인
  - 커스텀 스타일링

### Backend
- **Supabase**
  - PostgreSQL 데이터베이스
  - 실시간 데이터 동기화
  - REST API

### Deployment
- **Vercel**
  - 자동화된 배포
  - CI/CD 파이프라인

## 4. 데이터 모델

### plans 테이블

| 필드 | 타입 | 설명 |
|------|------|------|
| clerk_user_id | String | Clerk 사용자 식별자 |
| year | Integer | 연도 |
| month | Integer | 월(1-12) |
| content | String | 프로젝트 이름 |
| color | String | 색상 코드 |
| start_date | Date | 시작일 |
| end_date | Date | 종료일 |

## 5. MVP 기능 목록

### 1. 사용자 인증
- Clerk 기반 인증 시스템
- 소셜 로그인 (Google, GitHub)
- 이메일/비밀번호 로그인
- 사용자 프로필 관리

### 2. 기본 테이블 UI
- 12×N 그리드 레이아웃
- 프로젝트 시각화
- 색상 코딩

### 3. 프로젝트 CRUD
- 생성: 모달 폼
- 조회: 테이블 뷰
- 수정: 드래그 앤 드롭
- 삭제: 컨텍스트 메뉴

### 4. 실시간 동기화
- 멀티 디바이스 지원
- 실시간 UI 업데이트
